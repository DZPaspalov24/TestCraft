## 🌟 Project Overview

The project aims to develop an interactive electronic school application focused on a chosen subject, providing students with engaging educational content and tools for knowledge assessment.  

The application includes a total of **6 variations**, each containing **20 different questions**, designed to make learning more interactive and effective.
